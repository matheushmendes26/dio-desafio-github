Definição

HTML é um abriviação para Hyper Text Markup Language, que do inglês significa linguagem de marcação de hipertexto. Talvez pode haver algum tipo de confusão com a tradução mas o html não é uma lingua de programa e sim uma tipo de texto onde há marcadores ou tags que estruturam o texto onde são aplicados.

Início

Estruruta básica

Basicamente um arquivo html tem algumas divisões básicas, nas quais consistem: html, head, body.

Html

Todo o conteúdo que compõem o arquivo html é colocado dentro da tag <html> colocado entre sinais de maior que e menor que. da maneira como colocamos agora.
<html>
  <head>
   Congifurações...
  <head/>
  <body>
   Conteudo...
  <body/>
<html/>

Head

Já dentro da tag html é, sua subtag head é importante pois dá algumas configurações que são importante no momento de renderizar o arquivo html, como por exemplo o idioma padrão, links de arquivos a serem utilizados conjuntamente, a tag title que coloca o título na aba do navegador. Tirando a parte do title o conteúdo das outras tags em head não aparecerá no corpo do texto.

Body

É nesta tag onde serão colocados todo o contéudo que será visível ao usuário do vanegador. Boa parte das tags utulizadas serão colocadas dentro da tag body e onde a estruturação em html pode variar conforme a técnica e imaginação do desenvolvedor em questão.

Início

Tags HTML

As Tags no html são usadas para fazer a estruturação do documento como um todo. São por meio delas que será possível inserir variados tipos de conteúdos, como por exemplos textos, tabelas, listas, imagens, vídeos, formulários trechos de códigos entre outros.

Início

Atributos das Tags

Algumas tags podem vir acompanhadas por atributos. atributos são segmentaçãoes que são colocadas para fazer a estruturação e formatação de forma particularmente na tag que é associada. Alguns atributos são gerais e são utilizados conjuntamente com o CSS, já outros são atributos específicos para certas tags. Temos como exemplo os atributos "id", "class", "href" que são gerais, e por exemplo a tag input para inserir alguma entrada tem o atributo "type" que indica o tipo de entrada a ser inserida.

Início

Textos

Há várias tipos de tags que são utilizadas para estruturação de texto em particular. Podemos citar algumas segmentação, como por exemplo: p, strong, i, entre outras. Por meio delas é possível que o texto colocado seja melhor estrurutado, segmentado e com o auxílio do CSS, pode ser estilizado.

Início

Listas

Listas Ordenadas

São utilizadas para fazer lista numerdas. São utilizadas por meio da tag ol, e cada item é colocado com li. da senguinte forma.

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Listas não Ordenadas

São utilizadas para fazer lista não enumeradas. São utilizadas por meio da tag ul, e cada item é colocado com li. da senguinte forma.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Início

Links

Dentro das tags utilizadas, as tags para links são as mais importante dentro do html, por é por meio delas que há a navegação. a teg "a" vem da ancoragem que ancora o que é colocado a um local específico, seja da mesma págino ou da página seguinte por meio de um link, ou um identificador.
